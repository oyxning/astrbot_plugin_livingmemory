# WebUI 管理

LivingMemory 使用 AstrBot 官方插件 Pages 提供管理界面，不需要额外启动 Web 服务。

## 入口

进入 AstrBot WebUI：

`插件 -> LivingMemory -> Pages -> dashboard`

建议 AstrBot 版本为 `4.24.2` 或更高。旧版本如果没有官方 Pages 能力，插件仍可运行，但管理页面可能不可用。

## 页面能力

| 区域 | 用途 |
| --- | --- |
| 记忆管理 | 查看、搜索、删除长期记忆 |
| 召回调试 | 输入查询词，观察实际召回结果和排序 |
| 图谱视图 | 浏览实体、关系和记忆之间的连接 |
| 系统状态 | 查看索引、备份、统计数据和运行状态 |

## 图谱视图适合看什么？

图谱不是为了替代表格，而是帮助你发现记忆之间的结构关系：

| 观察点 | 示例 |
| --- | --- |
| 高频实体 | 用户、项目、地点、群聊主题 |
| 稳定关系 | “某人喜欢某事”“项目依赖某技术” |
| 跨记忆连接 | 多次对话中重复出现的同一实体 |
| 过期风险 | 很久没有被访问的低重要性关系 |

::: tip
WebUI 的操作会复用插件运行时的 MemoryEngine 和 GraphStore，不会绕过后端数据安全逻辑。
:::
